# Digit-Summation

> ML model that predicts the sum of MNIST digits in an image.

## How to run on ada

```
sinteractive -g 2
ssh -NfR 8888:localhost:8888 <username>@<ip address of your system> (ssh must be active for this to work)
jupyter notebook --no-browser
``` 
Go to localhost:8888 for seeing the notebook.
